Smart Assistant 사용법
=======================

🚀 빠른 시작
-----------
1. 더블클릭으로 실행: run_quick.bat
2. 터미널에서 실행: py quick_start.py

📧 이메일 포함 전체 테스트
--------------------------
1. 터미널에서 실행: py test_email.py
   (이메일 계정 정보를 .env에 설정한 뒤 실행)

🖥️ 전체 시스템 실행
-------------------
- GUI: py run_gui.py  
- CLI: py run_assistant.py

🔧 문제 해결
-----------
- python 명령어가 안 될 때: py 사용
- 한글 깨짐: UTF-8 인코딩 확인
- 로깅 오류: 간단한 스크립트 사용 (quick_start.py)

📁 주요 파일
-----------
- quick_start.py: 가장 간단한 테스트
- test_email.py: 이메일 포함 전체 테스트
- main.py: 메인 애플리케이션
- run_assistant.py: CLI 실행
- run_gui.py: GUI 실행
- run_quick.bat: 빠른 시작 배치 파일

🎯 실행 순서 권장
---------------
1. run_quick.bat (기본 동작 확인)
2. test_email.py (이메일 테스트)
3. run_gui.py 또는 run_assistant.py (전체 시스템)

✅ 성공 확인
-----------
다음과 같은 출력이 나오면 성공:
- ✅ 5개 메신저 메시지 수집
- ✅ 5개 메시지 요약 완료
- ✅ 5개 메시지 분류 완료
- ✅ 6개 액션 추출 완료
- 📋 TODO 리스트 생성 완료
