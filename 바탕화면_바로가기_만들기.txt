========================================
Smart Assistant 바탕화면 바로가기 만들기
========================================

🎯 추천 방법: Conda 전용 스크립트 사용
----------------------------------------
Anaconda/Miniconda를 사용하는 경우 (가장 확실함):

1. "Smart_Assistant_Conda.bat" 파일을 마우스 오른쪽 버튼으로 클릭
2. "바로 가기 만들기" 선택
3. 생성된 바로가기를 바탕화면으로 드래그
4. 바탕화면의 바로가기를 더블클릭하여 실행


방법 1: 자동 감지 스크립트 (일반)
----------------------------------------
1. "Smart_Assistant.bat" 파일을 마우스 오른쪽 버튼으로 클릭
2. "바로 가기 만들기" 선택
3. 생성된 바로가기를 바탕화면으로 드래그

* Python 환경을 자동으로 찾아서 실행합니다


방법 2: Anaconda Prompt에서 실행
----------------------------------------
자동 실행이 안 되는 경우:

1. "Open_Anaconda_Prompt_Here.bat" 더블클릭
2. 열린 Anaconda Prompt에서 다음 명령어 입력:
   python run_gui.py


방법 3: 콘솔 창 숨기기 (조용한 실행)
----------------------------------------
콘솔 창 없이 실행하려면:

1. "Smart_Assistant_Silent.vbs" 파일의 바로가기를 바탕화면에 만들기
2. 더블클릭하면 콘솔 창 없이 실행됩니다


선택사항: 아이콘 변경
----------------------------------------
- 바로가기를 마우스 오른쪽 버튼으로 클릭 → 속성
- "아이콘 변경" 버튼 클릭
- 원하는 아이콘 선택


문제 해결
----------------------------------------
Q: "Python 환경을 찾을 수 없습니다" 오류가 나요
A: "Smart_Assistant_Conda.bat" 또는 "Open_Anaconda_Prompt_Here.bat"를 사용하세요

Q: Conda 환경이 활성화되지 않아요
A: 시작 메뉴에서 "Anaconda Prompt"를 직접 열고
   cd C:\Users\USER\Desktop\OFFLINE-AGENT
   python run_gui.py
   명령어를 실행하세요

Q: 콘솔 창이 보기 싫어요
A: Smart_Assistant_Silent.vbs 파일을 사용하세요

Q: 여전히 실행이 안 돼요
A: 1. Anaconda Prompt 열기
   2. cd 명령어로 프로젝트 폴더로 이동
   3. python run_gui.py 실행


파일 설명
----------------------------------------
- Smart_Assistant.bat: 자동 감지 스크립트
- Smart_Assistant_Conda.bat: Conda 전용 스크립트 (권장)
- Smart_Assistant_Silent.vbs: 콘솔 창 없이 실행
- Open_Anaconda_Prompt_Here.bat: Anaconda Prompt 열기
